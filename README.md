# Text to Handwriting (Python Engine)

A beautiful, fully native Streamlit web application that converts your typed text into realistic handwriting.

## Features
- **Dynamic Word-Wrapping & Paging**: Automatically calculates text size to cleanly break lines and generate multiple A4 pages for you.
- **Multilingual Support**: Generate handwritten text in English, Hindi, Marathi, Nepali, Tamil, Malayalam, Bengali, Telugu, Gujarati, Kannada, Arabic, Russian, Chinese, Japanese, and Korean.
- **Humanizer Engine**: Introduces a customizable wobble and jitter to make the handwriting look authentically messy.
- **Custom Paper Styles**: Write on Ruled, Yellow Legal, Graph, or vintage Parchment backgrounds.
- **Draw Tool Integration**: Includes a signature/diagram canvas that gets natively stitched onto the end of your generated document.
- **1-Click Export**: Download all your generated pages directly as a ZIP file or a single PDF.

## How to Run Locally

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```

Enjoy!
